To set up the Recipe Manager locally, follow these steps:

1. Clone the repository:

```bash

```

2. Navigate to the project directory:

```bash
cd recipe-project
```

3. install dependencies:

    - install backend dependencies.

    ```bash
        cd backend
        npm install
    ```

    - install frontend dependencies.

    ```bash
        cd ../frontend
        npm install
    ```
